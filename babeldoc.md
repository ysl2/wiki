# babeldoc

> Ref: <https://github.com/funstory-ai/BabelDOC>

## Installation

```bash
uv tool install --python 3.12 BabelDOC

babeldoc --help
```

## Usage

```bash
babeldoc --openai --openai-model "deepseek-reasoner" --openai-base-url "https://api.deepseek.com" --openai-api-key "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" --files 'DeepSeek-AI et al. - 2024 - DeepSeek-V3 Technical Report.pdf'
```
