# 原码反码补码移码

1. 原码：
	正数的原码最高位（符号位）是0
	负数的原码最高位（符号位）是1
	其他的是数值位

	+7：	0 0000111
	-7: 	1 0000111
2. 反码：
	正数的反码与其原码相同，负数的反码与原码符号位不变，数值位取反
	就是1变0，0变1
	   +7：	0 0000111
	   -7：	1 1111000
3. 补码：
	正数的补码与原码相同
	负数的补码是在反码的基础上加1
	   +7：	0 0000111
	   -7：	1 1111001
	如果已知一个数x的补码，求-x的补码：
	把x的补码连着符号位一起取反，然后末位加1
4. 移码：符号位取反的补码。移码通常用于表示浮点数的阶码部分
5. 算术移位：
    算术移位的时候，小数点不动，数在动，数往左动是左移，数往右动是右移。
    算术移位的时候，前面的符号位不动，只动后面的真正的数字的部分。
    包括原反补的算术左移右移，补充规则：
    对于正数（符号位是0），原反补算术左右移全补0。
    对于负数（符号位是1），原码算术左右移全补0，反码算术左右移全补1，补码算术左移补0，算术右移补1。
6. 逻辑移位：针对无符号数，或者把有符号数看作无符号数。原反补逻辑左右移全补0。
