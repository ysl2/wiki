# woff2

> ttf <-> woff2

## Installation

```bash
git clone --recursive git@github.com:google/woff2.git
cd woff2
make clean all
mv woff2_* ~/.vocal
```

## Usage

```bash
wget https://ghfast.top/https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/FiraCode.zip
wget https://ghfast.top/https://github.com/laishulu/Sarasa-Term-SC-Nerd/releases/download/v2.3.1/SarasaTermSCNerd.ttf.tar.gz
```

```bash
woff2_compress FiraCodeNerdFont-Regular.ttf
```
